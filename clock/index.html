Well , Its a Clock. A plain , old looking, Simple Clock ( with hands ofcourse).

It's easy to configure and use , really.

add this to your html 

    <script src="{path_to_js}/clock.min.js"></script>
    <script>
            new Clock();
    </script>

and this is sufficient for clock to get started on your screen.

For more custom options, you can configure the clock by following parameters. 

new Clock({
     dimensions: 200,  // clock will be dimension x dimension in size
     needles: {        // object to customize the hand length for all three hands
         minutes: 75,
         hour: 60,
         seconds: 80 
     },
     img: "numbers_b"  // custom image to use in the background
});

The css used for the above clock

#clock {
    position: absolute;
    border: 5px solid;
    border-color: rgb(222, 184, 135);
    box-shadow: 2px 5px 5px 2px rgb(120,120,120);
    border-radius: 50px 0px 50px 0px;
    z-index: 10000;
    overflow: hidden;
}

After that, just enjoy your clock. See you another "TIME" :)
